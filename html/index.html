Introduction

The purpose of this project is to provide a few examples for retreiving data from the https://api-v3.mbta.com/ API

If you would like to setup the project locally you must clone the Git Repository . Steps for setup are included in the Readme of the repo. Otherwise, the application will continue to be hosted at https://mbta-route-planner.herokuapp.com/ for the next couple months

This web application is built on the NodeJS Express framework. I tried to keep all of the JS as vanilla as possible, using the axios npm package for making http requests and extending the body-parser middleware to parse http responses. The HTML interface also uses JQuery and the Handlebars templating engine to dynamically display data fetched from the API.
